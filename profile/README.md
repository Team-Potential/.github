# Team-Potential 

**Team-Potential**은 가능성을 현실로 만드는 열정적인 인디 게임 개발팀입니다.   

독창적인 아이디어와 탄탄한 기술력으로 새로운 게임 경험을 선사합니다.

우리의 이름처럼, 잠재력을 폭발시킬 준비가 되어 있습니다!
